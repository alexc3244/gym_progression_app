# Hypertrophy Engine

## Overview

This project is a personal hypertrophy training system that generates and adapts resistance training programs based on logged performance data.

It implements a hybrid progression model combining deterministic overload rules with autoregulation signals (RPE/RIR).

The system is designed as a lightweight decision engine rather than a full fitness application (for MVP dev, features to be built after MVP may expand scope)

## Goals

- Generate structured weekly hypertrophy programs
- Track workout execution and performance
- Adapt training loads using hybrid progression logic
- Maintain simplicity and interpretability over complexity

## Future-goals

- Machine learning / predictive modelling (for now)
- Full mobile application
- Features - AI/LM coach integration

## Non-Goals
- Nutrition tracking
- Wearable integration
- Social features
